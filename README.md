# 球球音乐

### 项目说明
>个人vue练习项目

* 技术栈:`vue+vue-router+vuex+stylus+node.js+nginx+element-ui+webpack`
* PS:因为自己域名还未通过备案,所以暂时无法预览。 

### 功能说明

>数据来源

数据来源于QQ音乐API,数据抓取详情请见blog文章[qq音乐数据的抓取](http://cxuyang.com/#/posts/jsonp)

>功能说明

* PC端(详情见QQ音乐主页,动画效果和数据都是一样的)
1. 主页展示
2. 我的音乐展示
3. 歌手页及其详情页展示
4. 分类歌单展示
5. 专辑页面展示
6. mv页面展示
7. 电台页面展示
8. 后续功能开发中...

* 移动端
1. 主页歌单展示
2. 歌手页及其详情展示
4. 排行榜展示
5. 播放器功能
6. 后续功能开发中...
